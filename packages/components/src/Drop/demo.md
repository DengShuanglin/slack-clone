## API

| 属性               | 类型              | 默认值   | 说明                                                  |
| :-----------     | :------          | :------- | :---------------------------------------------------- |
| backgroundColor  | string           | -        | 背景色                                                    |
| title            | string           | -        | 标题                                                    |
| fontSize         | string、number    | -        | 字体大小                                                   |
| width            | string、number    | -        | 输入框框度                                                   |
| height           | string、number    | -        | 输入框高度                                                   |
| color            | string            | -        | 字体颜色                                                   |
