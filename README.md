#### 特性简介

- 基于 React 的实时聊天系统
- 支持好友系统和一对一聊天，支持发送语音、图片和富文本消息
- 使用 monorepo 进行项目管理，确保了代码的可维护行和可复用性
- 采用 Vite 作为开发服务器，提升开发体验
